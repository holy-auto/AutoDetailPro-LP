# インフルエンサー アウトリーチ テンプレート集

`docs/influencer-research.md` で抽出した候補にコンタクトする際のDM/メールテンプレ。
カテゴリ・媒体・温度感（カジュアル／フォーマル）で使い分け可能。

> **重要**: 送信前に必ず（1）相手の直近投稿を1〜2本見て一言触れる、（2）名前を正確に入れる、（3）景表法・ステマ規制（2023年10月施行）に基づき #PR / #AD 表記を必須とする旨を明記する。

---

## 0. 共通方針

- **件名/冒頭は「あなたの○○を見て連絡しました」型**（業者っぽい一斉送信を避ける）
- **最初のDMでは商品説明を最小限**にし、「興味あれば詳細をお送りします」で1往復目を軽く
- **報酬・条件は2通目で出す**（最初に出すと値段交渉のフレームになる）
- **#PR / #AD 表記の明記**（景表法ステマ規制対応／必須）
- **NG投稿条件は契約段階で詰める**（最初のDMでは条件を盛り込みすぎない）

## 1. ファーストコンタクトDM（媒体別・カテゴリ別）

### 1-1. TikTok DM（カジュアル・短文）

```
はじめまして、AutoDetailPro（オートディテールプロ）の◯◯と申します。

@{アカウント名} さんの「{具体的な投稿/動画タイトル}」を拝見して、
{ペットとの車内シーン / お子さまとの車内 / キャンプ後の積載など}
にとても共感し、ご連絡いたしました。

私たちは「出張型カーディテーリング」のマッチングサービスを運営していて、
ご自宅の駐車場までプロが出張し、車内クリーニング〜外装コーティングまで
対応しています。

もしご興味があれば、{お住まいエリア}対応のプロをアサインして
"無料施工 + Before→After動画素材のご提供" をさせていただけないかと
思っております。

ご投稿は #PR 表記必須／内容はお任せ／辛口レビュー歓迎です。

詳細をお送りしてもよろしいでしょうか？
```

## 2. カテゴリ別「フック文」差し替え集

ファーストDMの3〜5行目（個別の共感パート）で使える定型フック。

### ペット系
- 「{ペットの名前}ちゃんとのドライブ動画、特に〇〇のシーンに癒やされました」
- 「大型犬を飼っていた経験があり、抜け毛と車内のニオイのリアルさに共感しました」

### ワーママ・子育て系
- 「お子さまが車内で食べこぼしされたエピソード、自分も同じ経験があり共感しました」
- 「チャイルドシート周りの汚れに本気で向き合っている発信が珍しく、お声がけしました」

### キャンプ・アウトドア系
- 「{ギア名}の積載動画、特に泥汚れの帰り道のリアルさに惹かれました」
- 「ハイエース系の車中泊コンテンツの中でも、生活感のあるリアルさが他と違うと感じました」

### ライフスタイル系
- 「家事代行を活用している投稿を拝見し、"車もプロに任せる"発想と相性が良いと感じました」

### 経営者・士業系
- 「"時間をお金で買う"発想に共感し、車関連の意思決定でもこの軸が活きると考えご連絡しました」

## 3. 詳細条件提示（2通目／返信あり後）

返信があったら送る、条件詰めの定型。

```
ご返信ありがとうございます。
詳細条件をお送りいたします。

■ サービス内容
- 出張型カーディテーリング（外装洗浄＋車内クリーニング＋簡易コーティング）
- 所要時間：2〜4時間（車のサイズ・状態により変動）
- 出張エリア：{対応エリア確認済}
- 通常価格：{¥xx,000} 相当 → 今回は無料

■ ご投稿条件
- 媒体：{TikTok / Instagram Reels / YouTube Shorts / フィード投稿} のいずれか1〜2本
- 公開タイミング：施工後14日以内
- 必須表記：#PR #AutoDetailPro
- 内容：構成・台本はお任せ／忌懚ないご感想OK
- NG事項：他の出張洗車サービスとの直接比較／施工士個人の批判表現

■ 撮影サポート
- Before/After 写真の撮影は施工士が対応可能
- 動画素材の提供（撮影オフショット）希望時は事前にご相談ください

■ 報酬
- 出張施工：無料
- 投稿フィー：{¥xx,000 / 応相談 / なし}
- ※プログラム経由予約発生時のレベニューシェア（成果報酬）も別途ご相談可

■ スケジュール
1. 本メールでご検討
2. ご合意の場合、契約書（電子）にご署名
3. 施工日程調整（2〜3週間以内目安）
4. 施工 → 投稿（14日以内）

ご質問・条件のすり合わせがあればお気軽にお知らせください。
```

## 4. フォローアップ（返信なし4〜7日後）

```
{アカウント名}さま

先日ご連絡させていただきました AutoDetailPro の{送信者}です。

ご多忙のところ恐れ入ります。
お忙しいところで埋もれてしまっていたら申し訳なく、念のため再送いたします。

ご興味の有無だけでもお知らせいただけますと幸いです。
（タイミングが合わない場合は、また別の機会にあらためてお声がけさせてください）
```

> 1回まで。2回目のフォローアップは送らない（迷惑行為になる）。

## 7. 法務／コンプラ チェックリスト（送信前）

- [ ] **景表法ステマ規制**：投稿条件に `#PR` または `#AD` 表記の必須化を明記
- [ ] **薬機法**：施工効果について「除菌100%」「アレルギー完治」等の医療的効能を匂わせない
- [ ] **個人情報**：相手の住所・車種・連絡先は契約締結後にのみ取得
- [ ] **著作権**：投稿された動画の二次利用範囲（公式サイト掲載可否等）を契約書側で明示
- [ ] **未成年**：18歳未満の場合は保護者同意必須＋夜間DM禁止

## 8. 運用ルール

- 同一カテゴリには **同時に最大3名**までしかアプローチしない（地域被り・露出時期被りを避けるため）
- 1日あたりのDM送信は **5件まで**（凍結リスク回避、特にInstagram）
- 全送信ログは下記で記録：

```
| 日付 | 媒体 | アカウント | テンプレ種別 | 担当 | 反応 | 次アクション |
|---|---|---|---|---|---|---|
|  |  |  |  |  |  |  |
```
