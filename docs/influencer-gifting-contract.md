# インフルエンサー ギフティング契約書（ドラフト）

> **重要・免責**
> このドキュメントは社内検討用の **ドラフト** です。実運用前に **必ず弁護士・法務専門家のレビュー** を受けてください。
> 特に景表法（ステマ規制）・薬機法・著作権・個人情報保護法・下請法（一部適用ケース）に関する条項は、契約スキームと実際の運用に照らして調整が必要です。

---

## 0. 契約スキームの整理（社内向けメモ）

| 項目 | 内容 |
|---|---|
| 契約形態 | 業務委託契約（成果物：SNS投稿）＋現物支給（出張施工サービス） |
| 当事者 | 甲：運営会社（株式会社Holy Auto／AutoDetailPro 運営）／乙：インフルエンサー（個人 or 法人） |
| 報酬性質 | 原則「現物（無料施工）」、追加で金銭フィー/レベニューシェアを設計可 |
| 想定下請法 | 乙が個人事業主かつ甲が資本金1,000万円超の場合は下請法対象 → 書面交付・60日以内支払い等の義務発生 |
| 源泉徴収 | 乙が個人で金銭フィーを支払う場合、原稿料/報酬として10.21%源泉徴収（要確認） |
| ステマ規制 | 2023/10/1施行・景表法 → 「広告であることが明瞭であること」の表示義務（甲が乙に表示させる責任を負う） |

## 1. 契約書本文（ドラフト）

```
インフルエンサー業務委託契約書

株式会社Holy Auto（以下「甲」という。）と、_______________________（以下「乙」という。）
は、AutoDetailPro（以下「本サービス」という。）の認知拡大を目的として、本日、
下記のとおり業務委託契約（以下「本契約」という。）を締結する。


第1条（目的）
本契約は、甲が乙に対し、本サービスを実際に体験する機会を無償で提供するとともに、
乙がその体験に基づき自らのSNSアカウント等で情報発信を行うにあたっての両者の
権利義務を定めることを目的とする。


第2条（甲の提供）
1. 甲は、乙に対し、別紙1に定める出張型カーディテーリングサービス
   （以下「本施工」という。）を無償で提供する。
2. 本施工の実施場所、日時、施工内容は、両者協議のうえ別途定める。
3. 甲は、本施工と併せて以下を提供する場合がある。
   (1) Before/After 撮影サポート
   (2) 制作物素材（ロゴ・サービス説明資料 等）
   (3) 別紙2に定める追加報酬


第3条（乙の業務）
1. 乙は、本施工の実施後 14 日以内に、別紙3に定める媒体・形式・本数にて
   本サービスに関する投稿（以下「本投稿」という。）を行う。
2. 本投稿には、以下のすべてを含めなければならない。
   (1) ハッシュタグ「#PR」または「#AD」「#プロモーション」のうちいずれか
       （景品表示法第5条第3号「一般消費者が事業者の表示であることを判別すること
       が困難である表示」の規制を回避するため、「広告であることが明瞭」となる
       位置・大きさで表示すること。投稿冒頭または冒頭近傍を推奨。）
   (2) ハッシュタグ「#AutoDetailPro」
   (3) 本サービスへの導線（プロフィールリンク／概要欄リンク／ストーリーズリンク
       のいずれか）
3. 乙は、本投稿の構成・台本・編集について、自らの判断で行うことができる。
   ただし、第4条（禁止事項）に抵触してはならない。


第4条（禁止事項）
乙は、本投稿および関連する発信において、以下の行為を行ってはならない。
(1) 本サービスについて事実と異なる表示をすること
(2) 効能効果について「除菌」「アレルギー」「治癒」等、医療・医薬品的な表現を行うこと
    （薬機法第66条等に抵触する可能性があるため）
(3) 個別の競合サービスを名指しで批判または比較すること
(4) 施工担当プロ個人を特定したうえでの誹謗中傷
(5) 第三者の権利（著作権、肖像権、プライバシー権、商標権等）を侵害すること
(6) 公序良俗に反する内容、差別的・暴力的・性的に過激な表現
(7) 本サービスの未公開情報、価格体系の詳細条件、内部資料を公開すること


第5条（広告該当性の表示／ステマ規制対応）
1. 甲乙は、本投稿が景品表示法令和5年告示第19号
   （「一般消費者が事業者の表示であることを判別することが困難である表示」）
   に該当する「事業者の表示」であることを相互に確認する。
2. 乙は、本投稿につき、第3条第2項(1)号に定めるハッシュタグまたはこれに相当する
   明示（例：「【PR】」「タイアップ投稿」等の冒頭表記）により、
   一般消費者が事業者の表示であることを容易に判別できる方法で表示する。
3. 甲は、乙の表示状況を投稿後速やかに確認し、不足がある場合は乙に修正を依頼できる。


第6条（事前確認）
1. 乙は、本投稿の公開前に、甲に対し原稿または完成データを共有し、第4条の
   禁止事項および第5条の表示要件への適合確認を求める。
2. 甲は、共有を受けてから 3 営業日以内に確認結果を乙に通知する。期限内に
   通知がない場合、適合とみなす。
3. 甲の確認は、第4条・第5条への適合性に限定され、表現・構成への介入は行わない。


第7条（著作権・利用許諾）
1. 本投稿および関連する写真・動画素材（以下「本コンテンツ」という。）の著作権は、
   原始的に乙に帰属する。
2. 乙は、甲に対し、本コンテンツを以下の目的で無償・無期限・非独占的に利用する
   ことを許諾する。
   (1) 甲の運営する公式ウェブサイト・LP・SNSアカウントでの掲載・転載
   (2) 甲のサービス紹介資料、営業資料、プレス資料への掲載
   (3) 甲が出稿する広告クリエイティブへの利用
3. 甲は、前項の利用にあたり、出典として乙のアカウント名を明示する。
4. 甲は、本コンテンツの編集・トリミングを行う場合、乙の人格的利益を害さない
   範囲で行うものとする。改変が乙の意に反する重大な変更となる場合は、
   事前に乙の同意を得る。


第8条（投稿の維持）
1. 乙は、本投稿を公開日から 12ヶ月以上維持するよう努める。
2. 乙の判断によりやむを得ず削除する場合は、事前に甲へ通知する。


第9条（個人情報）
1. 甲は、本契約の履行のため必要な範囲で乙の個人情報（氏名、住所、車両情報、
   連絡先、振込口座等）を取得・利用する。
2. 甲は、当該個人情報を第三者に提供しない。ただし、本施工を実施する施工担当
   プロ（甲のプラットフォーム登録事業者）に対しては、施工実施に必要な範囲で
   提供することを乙は同意する。


第10条（報酬）
1. 第2条第1項の本施工提供をもって、乙への基本対価とする。
2. 別紙2にて追加の金銭報酬を定める場合、甲は本投稿の公開確認後 30 日以内
   （ただし下請法適用時は本投稿の検収完了後 60 日以内のいずれか早い日）に
   乙が指定する口座へ振込する。振込手数料は甲負担。
3. 源泉徴収義務がある場合、甲は法令に基づき源泉徴収を行ったうえで支払う。


第11条（第三者からのクレーム）
1. 本投稿に関し第三者から権利侵害等の申立てを受けた場合、甲乙は速やかに相互に
   通知し、誠実に協議のうえ対応する。
2. 一方当事者の故意または重過失に起因する場合、当該当事者は他方当事者に
   生じた損害を賠償する。


第12条（契約期間・解除）
1. 本契約の有効期間は契約締結日から本投稿公開後 12ヶ月とする。
2. 一方当事者が本契約に違反し、相当期間を定めた催告後も是正しない場合、
   他方当事者は本契約を解除できる。
3. 解除された場合、甲は乙に対し本施工相当額（別紙1記載額）の返還を
   請求できる。ただし、乙の責に帰すべき事由がない場合はこの限りでない。


第13条（秘密保持）
甲乙は、本契約の履行に関連して知り得た相手方の業務上・営業上の情報のうち、
公知のものを除き、第三者に開示・漏洩してはならない。本条項は契約終了後も
3年間有効とする。


第14条（協議事項）
本契約に定めなき事項または疑義が生じた事項については、甲乙が誠実に協議の
うえ解決する。


第15条（合意管轄）
本契約に関する紛争は、東京地方裁判所を第一審の専属的合意管轄裁判所とする。


本契約締結の証として、本書 2 通を作成し、甲乙記名押印または電子署名のうえ、
各 1 通を保有する。
```

## 4. 既存契約との関係

このドラフトは下記前提で書かれています。実際の運用と齟齬がないかチェック：

- [ ] 「施工担当プロ」は甲のマッチングプラットフォーム登録事業者であり、
      甲とは別途の業務委託関係を有している（このため本契約では乙→甲のみで完結）
- [ ] プロ側の個別氏名・連絡先は本契約締結後にのみ乙へ開示
- [ ] 乙の個人情報をプロへ提供する旨の同意条項（第9条第2項）が
      既存のプライバシーポリシーと整合している

## 5. 法的レビューでの主な確認ポイント

弁護士レビュー時に明示的に確認したい論点：

1. **景表法ステマ規制への対応十分性**
   - ハッシュタグ表記のみで「一般消費者が判別容易」と認められるか
   - ストーリーズ・短尺動画でのスタンプ表記・文字盤表記の十分性
2. **薬機法に抵触する表現の具体的範囲**
   - 「除菌」「抗ウイルス」等の表現がOK／NGの境界
3. **下請法該当性**
   - 甲の資本金規模と乙の属性次第。該当時は別途追記事項あり（書面の即時交付・支払期日60日以内・受領拒否禁止等）
4. **著作権の利用許諾範囲（第7条）**
   - 「無償・無期限」が妥当か、有期にすべきか
   - 二次的著作物作成権をどこまで認めるか
5. **投稿削除時の取り扱い（第8条）**
   - 努力義務にとどめるか、確定義務として違約金を定めるか
6. **契約解除時の施工費返還（第12条第3項）**
   - 何割を返還対象とするか、過大な違約金とみなされないか
